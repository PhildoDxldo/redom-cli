# RE:DOM project generator
Initialize [RE:DOM](https://redom.js.org) projects easily!

## 1. Install
```
$ npm install -g redom-cli
```

## 2. Create project
```
$ redom
```

## 3. Develop!
```
$ cd [project name]
$ npm run dev
```
- Open [http://localhost:8080](http://localhost:8080)

## Tip
To define path to install to, use
```
$ redom my-project
```
